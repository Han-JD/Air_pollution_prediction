# Air_pollution_prediction
Predict Air pollution at backyoungdo base on the data from the beijing, shanghai, and shenyang.

Random forest<br/>
Accuracy: 0.8031<br/>
Precision: 0.7794<br/>
Recall: 0.8031<br/>
F1: 0.7416<br/>
At 5 hours time difference.
